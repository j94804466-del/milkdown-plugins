# Milkdown Plugins

一组 [Milkdown](https://milkdown.dev) 编辑器的扩展插件。

## 📦 包列表

| 包名 | 描述 |
|------|------|
| [@xz-summer/milkdown-container](./packages/container) | 容器指令插件（info、tip、warning、danger、details） |
| [@xz-summer/milkdown-slash-menu-core](./packages/slash-menu-core) | 斜杠菜单核心（框架无关） |
| [@xz-summer/milkdown-slash-menu-react](./packages/slash-menu-react) | 斜杠菜单 React 版 |
| [@xz-summer/milkdown-slash-menu-vue](./packages/slash-menu-vue) | 斜杠菜单 Vue 版 |

## 🚀 安装

```bash
# 容器插件
pnpm add @xz-summer/milkdown-container

# 斜杠菜单（React）
pnpm add @xz-summer/milkdown-slash-menu-react

# 斜杠菜单（Vue）
pnpm add @xz-summer/milkdown-slash-menu-vue
```
